Adding the ShitRust Icon as a Favicon

To add the ShitRust icon as a favicon to your website, include the following HTML in your document's <head> section:

<!-- Basic favicon -->
<link rel="icon" href="icon.png" type="image/png">

<!-- Apple Touch Icons -->
<link rel="apple-touch-icon" sizes="180x180" href="apple-touch-icon.png">

<!-- Favicon for modern browsers -->
<link rel="icon" type="image/png" sizes="32x32" href="favicon-32x32.png">
<link rel="icon" type="image/png" sizes="16x16" href="favicon-16x16.png">

<!-- Microsoft Tile -->
<meta name="msapplication-TileColor" content="#d16969">
<meta name="msapplication-TileImage" content="mstile-144x144.png">

<!-- Safari Pinned Tab Icon -->
<link rel="mask-icon" href="safari-pinned-tab.svg" color="#d16969">

<!-- Browser Theme Color -->
<meta name="theme-color" content="#2b2b2b">
